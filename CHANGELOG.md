## [1.0.0]

### Added
- Full frame parser for all 9 MR60BHA2 frame types (including 3 undocumented)
- Embedded HTTP + WebSocket server on ESP32C6
- WebSocket JSON schema compatible with radar-dash
- WiFi credentials via Kconfig (no credentials in source)
- OTA firmware update support with automatic rollback
- PROTOCOL.md documenting the complete binary UART protocol
